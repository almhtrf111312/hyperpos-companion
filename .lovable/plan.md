

# تحسين تصميم إعدادات المتجر - تقليل المساحة المستخدمة

## المشكلة
صفحة "إعدادات المتجر" تستهلك مساحة عمودية كبيرة جدا على الهاتف المحمول بسبب:
1. كل حقل (اسم المتجر، الهاتف، البريد...) يأخذ سطرين: واحد للعنوان وواحد للحقل
2. أسعار الصرف تأخذ مساحة ضخمة (بطاقتان كبيرتان بـ 4 حقول)
3. قسم اللغة يستخدم Card كامل مع header وdescription
4. خيارات نقطة البيع (إخفاء الصيانة) تأخذ مساحة أكبر من اللازم

## الحل

### 1. حقول معلومات المتجر - تصميم مدمج (inline)
تحويل كل حقل من تصميم عمودي (label فوق + input تحت) الى تصميم أفقي مدمج:
- العنوان (label) على اليمين والحقل (input) على اليسار في نفس السطر
- هذا يوفر سطر كامل لكل حقل (5 حقول = توفير 5 أسطر)

```text
قبل:                          بعد:
┌──────────────────┐          ┌──────────────────────────┐
│ اسم المتجر       │          │ اسم المتجر │ [________] │
│ [______________] │          │ نوع المحل  │ [________] │
│ نوع المحل        │          │ الهاتف     │ [________] │
│ [______________] │          │ البريد     │ [________] │
│ الهاتف           │          │ العنوان    │ [________] │
│ [______________] │          └──────────────────────────┘
│ البريد           │
│ [______________] │
│ العنوان          │
│ [______________] │
└──────────────────┘
```

### 2. أسعار الصرف - تصميم مضغوط
تحويل من بطاقتين كبيرتين (كل واحدة بحقلين منفصلين) الى جدول مضغوط:
- كل عملة في صف واحد: [اسم العملة] [سعر الصرف]
- العملية الحسابية أسفل كل صف بخط صغير

```text
قبل:                              بعد:
┌─────────────────────┐          ┌─────────────────────────────┐
│ ■ اسم العملة الأولى │          │ أسعار الصرف                 │
│ [________________]  │          │ [ليرة تركية] [32        ]   │
│ سعر الصرف           │          │  1$ = 32 ليرة تركية         │
│ [________________]  │          │ [ليرة سورية] [14500     ]   │
│ 1$ = 32 ليرة تركية  │          │  1$ = 14500 ليرة سورية      │
├─────────────────────┤          └─────────────────────────────┘
│ ■ اسم العملة الثانية│
│ [________________]  │
│ سعر الصرف           │
│ [________________]  │
│ 1$ = 14500 ل.س      │
└─────────────────────┘
```

### 3. قسم اللغة - تصميم مضغوط
تحويل LanguageSection من Card كامل (مع CardHeader وCardDescription) الى صف بسيط:
- عنوان "اللغة" على اليمين وقائمة الاختيار على اليسار

### 4. خيار إخفاء الصيانة - تصغير
تقليل padding وإزالة الوصف الإضافي، ليصبح سطر واحد مدمج مع Switch
