
# إضافة 3 لغات جديدة (تركي، فارسي، كردي) + محدد اللغة في شاشة الدخول

## المرحلة 1: البنية التحتية والترجمات

### 1. تحديث نوع اللغة والإعدادات (`src/lib/i18n.ts`)
- تعديل `Language` type ليصبح: `'ar' | 'en' | 'tr' | 'fa' | 'ku'`
- إضافة 3 لغات جديدة لمصفوفة `languages`:
  - `{ code: 'tr', name: 'Turkish', nativeName: 'Türkçe', direction: 'ltr' }`
  - `{ code: 'fa', name: 'Persian', nativeName: 'فارسی', direction: 'rtl' }`
  - `{ code: 'ku', name: 'Kurdish', nativeName: 'کوردی', direction: 'rtl' }`
- تحديث `getCurrentLanguage()` للتعرف على الأكواد الجديدة
- إضافة كتل الترجمة الكاملة لكل لغة (tr, fa, ku) بنفس المفاتيح الموجودة (~700 مفتاح لكل لغة)

### 2. تحديث hook اللغة (`src/hooks/use-language.tsx`)
- إضافة `'fa'` و `'ku'` لمصفوفة `RTL_LANGUAGES`
- تحديث دالة `mapSystemLanguage` في `src/lib/system-language.ts` للتعرف على `'tr'`, `'fa'`, `'ku'`

### 3. إضافة محدد اللغة في شاشة الدخول والتسجيل
- إضافة مكون `LanguageQuickSelector` صغير في أعلى صفحة Login وSignup
- شكله: زر/تبويب صغير يعرض اسم اللغة الحالية بلغتها الأصلية (مثلاً "العربية")
- عند النقر: يفتح قائمة منسدلة بجميع اللغات المتاحة
- عند اختيار لغة: يتغير التطبيق فوراً بدون زر حفظ

### الملفات المتأثرة

| الملف | التعديل |
|-------|---------|
| `src/lib/i18n.ts` | تعديل Type + إضافة ~4500 سطر ترجمة (3 لغات x ~1500 مفتاح) |
| `src/hooks/use-language.tsx` | إضافة RTL languages |
| `src/lib/system-language.ts` | إضافة mapping للغات الجديدة |
| `src/pages/Login.tsx` | إضافة محدد اللغة في الأعلى |
| `src/pages/Signup.tsx` | إضافة محدد اللغة في الأعلى |
| `src/components/settings/LanguageSection.tsx` | سيعمل تلقائياً لأنه يقرأ من مصفوفة `languages` |

### ملاحظة مهمة
نظراً لحجم الترجمات الكبير (~700 مفتاح x 3 لغات)، سيتم تقسيم التنفيذ:
1. أولاً: البنية التحتية + محدد اللغة + الترجمات الأساسية (التنقل، المصادقة، الأزرار المشتركة)
2. ثانياً: ترجمة بقية الأقسام (المنتجات، العملاء، الفواتير، الإعدادات، إلخ)

### شكل محدد اللغة في شاشة الدخول

```text
+----------------------------------+
|  [العربية v]     <-- أعلى يمين   |
|                                  |
|     [Logo]                       |
|     مرحباً بك في FlowPOS Pro    |
|     ...                          |
+----------------------------------+
```

محدد بسيط بدون عنوان، فقط اسم اللغة الحالية. النقر يفتح القائمة ويغير اللغة فوراً.
